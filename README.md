# Simple Interest Calculator (Bash)

This project is a simple interest calculator written in Bash. It prompts the user for:
- Principal amount (P)
- Rate of interest (R)
- Time period in years (T)

It then calculates Simple Interest using:
SI = (P * R * T) / 100

## How to Run
1. Make the script executable:
   chmod +x simple-interest.sh
2. Run it:
   ./simple-interest.sh
